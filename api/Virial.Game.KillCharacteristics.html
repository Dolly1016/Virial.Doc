Enum KillCharacteristics

Virial.Game
[Flags]
public enum KillCharacteristics

Fields

Disappear = 0

キルされそうになると煙になって消滅します。

FlagKillRealPlayer = 2

実際のプレイヤーをキルします。

FlagLeftDeadBody = 4

足元に死体を発生させます。 死体を残さないキルであればこのフラグは無視されます。

FlagLeftRealDeadBody = 8

実際のプレイヤーの足元に死体を発生させます。 死体を残さないキルであればこのフラグは無視されます。

KillAllAndLeaveBodyAll = KillAllAndLeaveBodyOne | FlagLeftRealDeadBody

自身と本体もろとも死亡し、全て死体を発生させます。 死体を発生させないキルであれば死体は発生しません。

KillAllAndLeaveBodyOne = FlagKillRealPlayer | FlagLeftDeadBody

自身と本体もろとも死亡し、死体は自身の元にのみ残します。 死体を発生させないキルであれば死体は発生しません。

KillOne = 4

自身が死体になります。自身が本体でない限り本体は死亡しません。
