Namespace Virial.Attributes

Classes

AddonDocumentAttribute

ID付きドキュメントを表すクラスです。

EventPriority

リスナに優先度を設定します。デフォルトの優先度は100で、値が大きいほど優先して実行されます。

Local

ローカルでのみ呼び出される手続きを表します。 IBindPlayerを実装するクラスで定義したリスナでAmOwnerがtrueの場合のみ実行されます。

NebulaPreprocess

プリプロセスメソッドを実装しているクラスにこの属性を付加します。 静的な公開メソッドPreprocess(INebulaPreprocessor)を実装していると記述された処理が実行されます。 そうでない場合でも静的コンストラクタが走ります。

NebulaRPC

メソッド呼び出しをRPC呼び出しに置き換える属性です。

OnlyHost

ホストでのみ呼び出される手続きを表します。

OnlyLocalPlayer

GameOperatorに付与できる属性です。 イベントがAbstractPlayerEventを継承している場合にのみ使用できます。 イベントのPlayerのAmOwnerがtrueの場合のみ実行されます。

OnlyMyPlayer

GameOperatorに付与できる属性です。 IBindPlayerを実装するクラスで定義したリスナでイベントがAbstractPlayerEventを継承している場合にのみ使用できます。 MyPlayerとイベントのPlayerが一致する場合のみ実行されます。

RequiringHandshake

呼び出しによってハンドシェイクが求められるメソッドであることを示す属性です。
